Average
Calculating Average speed for 2 different speeds with same distance

If a person travels the same distance with 'a' km/hr and 'b' km/hr then average speed will be calculated as:

$$\frac{2ab}{a+b}km/hr$$

Calculating Average speed for 3 different speeds with same distance

If a person travels the same distance with 'a' km/hr, 'b' km/hr and 'c' km/hr, then average speed will be calculated as:

$$\frac{3abc}{ab+bc+ca}km/hr$$
